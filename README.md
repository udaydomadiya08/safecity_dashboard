How to run locally

Unzip and cd into the project folder.

(Optional) create & activate a virtualenv:

    python3 -m venv venv

(Linux/macOS)

    source venv/bin/activate 
(Windows)

    venv\Scripts\activate 
then:

    pip install -r requirements.txt

    python app.py

Open http://127.0.0.1:5000 in your browser
